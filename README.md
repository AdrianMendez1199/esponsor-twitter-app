# esponsor-twitter-app

## Requirements

```sh
docker -v 
    Docker version 20.10.5
```

## Getting started

* clone project 
  ```sh 
     git clone project-url and git submodule update --init 
   ``` 
* copy .env.sample .env
* docker-compose up -d --build
* Backend running on http://127.0.0.1 and frontend http://127.0.0.1:3000
